Telemetry

Content Attribution Telemetry
SDK Available January 2026

Track What AI Systems Actually Use

OpenAttribution Telemetry is a session-based standard for tracking content events, conversation turns, and outcomes in AI systems. It provides the audit trail needed for licensing verification, publisher compensation, and compliance.

Content Events

Track when content is retrieved, displayed, cited, or engaged with.

Conversation Tracking

Record conversation turns with configurable privacy levels.

Commerce Outcomes

Link content usage to outcomes: conversions, purchases, engagement.

Telemetry works alongside AIMS (agent identity) and links sessions to agent manifests via DID references.

Session Model

A Session represents a bounded interaction between a user and an AI agent:

Session
├── session_id (UUID)
├── started_at (timestamp)
├── content_scope (which content mix/sources available)
├── manifest_ref (AIMS DID reference)
├── user_context (segments, attributes — privacy-controlled)
│
├── events[]
│   ├── content_retrieved   — Content fetched from source
│   ├── content_displayed   — Content shown to user
│   ├── content_cited       — Content referenced in response
│   ├── content_engaged     — User interacted with content
│   ├── turn_completed      — Conversation turn finished
│   └── [commerce events]   — product_viewed, cart_add, checkout, etc.
│
├── ended_at (timestamp)
│
└── outcome
    ├── type (conversion | browse | abandonment)
    ├── value_amount (if conversion)
    └── currency

Event Types

Content Events

Event When to Emit Key Fields
content_retrieved Content fetched from source content_id, source_uri, retrieval_method
content_displayed Content shown to user content_id, display_position
content_cited Content referenced in response content_id, citation_context
content_engaged User interacted with content content_id, engagement_type

Conversation Events

Event When to Emit Key Fields
turn_started User initiated a turn query_intent (if disclosed)
turn_completed Agent finished responding content_ids_cited, response_tokens

Commerce Events

Event When to Emit Key Fields
product_viewed Product page shown product_id, source_content_id
product_compared Products compared product_ids
cart_add Added to cart product_id, quantity
checkout_started Checkout initiated cart_value
checkout_completed Purchase completed order_value, currency
checkout_abandoned Checkout abandoned cart_value

Privacy Levels

Telemetry supports configurable privacy to accommodate different trust relationships:

Level Query Text Response Text Intent Topics Content IDs
full Yes Yes Yes Yes Yes
summary Summarized Summarized Yes Yes Yes
intent No No Yes Yes Yes
minimal No No No No Yes

Choosing a Privacy Level

  • full: First-party analytics where you control both agent and telemetry
  • summary: Trusted partners who need context but not verbatim content
  • intent: Publisher compensation (they see what content was used, not user queries)
  • minimal: Compliance verification only (prove content was used, nothing more)

Quick Start

Installation

pip install openattribution-telemetry

Basic Usage

from openattribution.telemetry import Client, ConversationTurn, SessionOutcome

async with Client(
    endpoint="https://telemetry.example.com",
    api_key="your-api-key",
) as client:

    # Start a session
    session_id = await client.start_session(
        content_scope="shopping-assistant",
        manifest_ref="did:aims:web:example.com:shopping-assistant",
    )

    # Record content retrieval
    await client.record_event(
        session_id=session_id,
        event_type="content_retrieved",
        content_id=content_id,
        metadata={
            "source_uri": "https://wirecutter.com/headphones",
            "retrieval_method": "content_mixer",
        }
    )

    # Record a conversation turn
    await client.record_event(
        session_id=session_id,
        event_type="turn_completed",
        turn=ConversationTurn(
            privacy_level="intent",
            query_intent="product_recommendation",
            content_ids_cited=[content_id],
        )
    )

    # End session with outcome
    await client.end_session(
        session_id=session_id,
        outcome=SessionOutcome(
            type="conversion",
            value_amount=14999,
            currency="USD",
        )
    )

Framework Integrations

Telemetry integrates with popular AI agent frameworks.

LangChain Callback Handler

from langchain.callbacks.base import BaseCallbackHandler
from openattribution.telemetry import Client

class AttributionCallbackHandler(BaseCallbackHandler):
    def __init__(self, client: Client, session_id):
        self.client = client
        self.session_id = session_id

    async def on_retriever_end(self, documents, **kwargs):
        for doc in documents:
            await self.client.record_event(
                session_id=self.session_id,
                event_type="content_retrieved",
                content_id=doc.metadata.get("id"),
                metadata={"source_uri": doc.metadata.get("source")}
            )

LlamaIndex Span Handler

from llama_index.core.instrumentation.span_handlers import BaseSpanHandler
from openattribution.telemetry import Client

class AttributionSpanHandler(BaseSpanHandler):
    def __init__(self, client: Client, session_id):
        self.client = client
        self.session_id = session_id

    def handle_span_end(self, span):
        if span.span_type == "retrieval":
            for node in span.nodes:
                self.client.record_event(
                    session_id=self.session_id,
                    event_type="content_retrieved",
                    content_id=node.metadata.get("doc_id"),
                )
