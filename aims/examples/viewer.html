About This Example

This manifest demonstrates how an AI provider would declare their system's provenance, tuning, and capabilities. It uses W3C Verifiable Credentials format and includes:

  • Training data composition and sources
  • Fine-tuning datasets and methods
  • Runtime access to tools and licensed content
  • Cryptographic signatures for verification
